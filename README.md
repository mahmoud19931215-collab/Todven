# Togven – متجر إلكتروني حديث

- عرض سريع جداً بفضل التخزين المحلي (IndexedDB)
- وضعين للعرض: Hero (كبير) و List (قائمة)
- دعم الوضع الليلي
- إرسال الطلبات عبر واتساب

## المتطلبات
- رابط Google Apps Script يعيد بيانات JSON بالصيغة:
{
  "categoryName": [
    { "name": "product", "price": 1000, "imageUrl": "url", "stock": 10 }
  ]
}
